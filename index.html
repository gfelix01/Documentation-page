Java punto de entrada

El punto de incio o entrada en JAVA es con la Declaracion de la funcion Main

Esta caracterizado por un (String[]arg) como parametro de la funcion 

                
                    public class HolaMundo {
                        public static void main(String[] args) {
                          System.out.println("Hola Mundo!");
                        }
                      }

La función (main) acepta un parámetro de tipo String que se utiliza para pasar argumentos de línea de comando al programa. Estos argumentos son opcionales y pueden ser utilizados para personalizar la ejecución del programa.

Programacion Orientada a Objetos

La programación orientada a objetos (POO) es un paradigma de programación que se basa en la idea de organizar el código en "objetos" que encapsulan datos y comportamientos. En la POO, los objetos son instancias de una clase, que define las propiedades y métodos que los objetos de esa clase tendrán.

La POO se centra en la definición de objetos, que son entidades que tienen un estado interno y un comportamiento. Los objetos interactúan entre sí mediante el envío de mensajes, que son solicitudes para realizar una operación específica. Cada objeto tiene un conjunto de atributos, que son las características que definen su estado, y métodos, que son las operaciones que puede realizar.

La POO tiene varias ventajas, como la modularidad, la reutilización de código y la capacidad de representar de manera más precisa el mundo real. Además, permite la creación de aplicaciones más grandes y complejas, ya que se pueden dividir en objetos más pequeños y manejables. 

                        
                            public class Coche {
                                // Variables de instancia
                                private String marca;
                                private String modelo;
                                private int ano;
                                
                                // Constructor
                                public Coche(String marca, String modelo, int ano) {
                                    this.marca = marca;
                                    this.modelo = modelo;
                                    this.ano = ano;
                                }
                            }

En este ejemplo, la clase Coche tiene tres variables de instancia (marca, modelo y ano), un constructor que inicializa estas variables, tres métodos (arrancar, frenar y getAno) y un método principal (main) que crea una instancia de Coche, llama a sus métodos y muestra información sobre el año del coche.
